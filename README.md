# SiamFIARPNpp

## Compile
```
cd  SiamRPNpp
python setup.py build_ext --inplace
```

```
Ubuntu: 20.04
Python: 3.8
CUDA: 11.3

```

```
cd  SiamRPNpp
python ./bin/my_train.py
```

```
cd  /SiamRPNpp

```
## Eval
```
cd  SiamRPNpp
python ./bin/my_eval.py
```
